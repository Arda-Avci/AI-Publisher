## 🔜 Yapılacaklar (sadece test)

### 1. Sandbox API Key Girişi
Kullanıcı `.env` dosyasına şunları ekleyecek:
```
IYZICO_API_KEY=sandbox-...
IYZICO_SECRET_KEY=sandbox-...
IYZICO_BASE_URL=https://sandbox-api.iyzipay.com
```
